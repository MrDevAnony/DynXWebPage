                            Clear DNS Cache
                        

                    

                    

                        

                            Windows
                            

                            Open CMD and enter ipconfig /flushdns
                            



                            Linux (systemd-resolved)
                            

                            Open Terminal and enter
                            sudo resolvectl flush-caches
                            

                            or
                            

                            sudo systemctl restart systemd-resolved
                            



                            macOS
                            

                            Open Terminal and enter
                            sudo killall -HUP mDNSResponder
                            

                            (for older versions use
                            sudo dscacheutil -flushcache)
                            



                            Linux (nscd)
                            

                            Open Terminal and enter
                            sudo service nscd restart
                            

                            or
                            

                            sudo /etc/init.d/nscd restart
                            



                            Linux (dnsmasq)
                            

                            Open Terminal and enter
                            sudo service dnsmasq restart
                            

                            or
                            

                            sudo /etc/init.d/dnsmasq restart
                            



                            Google Chrome
                            

                            Type chrome://net-internals/#dns in the
                            address bar and click Clear host cache
                            



                            Mozilla Firefox
                            

                            Type about:networking#dns in the
                            address bar and click Clear DNS Cache
                            



                            Opera
                            

                            Type opera://net-internals/#dns in the
                            address bar and clear the cache 



                            Safari
                            

                            Use macOS Terminal command (see macOS instructions)
                            as Safari uses system DNS cache
                            



                            Android
                            

                            Toggle Airplane mode or restart the device
                            



                            iOS
                            

                            Toggle Airplane mode or restart the device
                            



                            Routers
                            

                            Restart the router to clear its DNS cache
